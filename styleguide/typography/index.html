Typography

Typography describes the fonts, heading size relationships, line and character spacing and styling of specific HTML textual elements.

OpenSans is currently being used.

Headings

MyService uses the DTA's default UI-Kit heading styles with some additional font-size and font-weight overrides. All HTML headings, <h1> through <h6>, are available.

Body

MyService uses the DTA's default UI-Kit body styles. This is applied to the <body>, ordered and un-ordered lists, and all paragraphs.

  2. item 2
  3. item 3

  • item 3
PARAGRAPH EXAMPLE

This is an example of paragraph text. Note that when the paragraph reaches a certain width on the page, it wraps to a new line. This is set by UI-Kit using the max-width parameter.
